# 53. 最大子数组和

## 题目信息

- 难度：中等
- 链接：https://leetcode.cn/problems/maximum-subarray/
- 主知识点：动态规划
- 相关知识点：数组、线性 DP、Kadane 算法、分治
- 复习状态：`new`

## 一眼识别

看到这些信号时，应该想到本题方法：

- 要找的是连续子数组，不是任意子序列
- 目标是最大和
- 每个位置都可以选择“接在前面的连续子数组后面”或者“从自己重新开始”
- 只需要最大和，不需要返回具体区间时，可以用 O(1) 空间维护状态

## 解题思路

### 暴力思路

可以枚举所有子数组的左右边界，再计算每个子数组的和。朴素做法是 O(n^3)，用前缀和可以降到 O(n^2)，但 `nums.length` 最大是 `10^5`，仍然不够好。

### 优化关键

定义状态：

```text
sum 表示以 nums[i] 结尾的最大连续子数组和
```

当遍历到 `nums[i]` 时，以 `i` 结尾的最大子数组只有两种选择：

- 接上前一个位置结尾的最大子数组：`sum + nums[i]`
- 不接前面，从当前元素重新开始：`nums[i]`

所以状态转移是：

```cpp
sum = max(sum + nums[i], nums[i]);
```

全局答案不一定以最后一个位置结尾，因此还需要 `maxAns` 记录遍历过程中出现过的最大 `sum`。

### 最终做法

1. 初始化 `sum = 0`
2. 初始化 `maxAns = nums[0]`，保证全负数数组也能得到正确答案
3. 从左到右遍历数组
4. 更新以当前位置结尾的最大子数组和 `sum`
5. 用 `sum` 更新全局最大值 `maxAns`
6. 返回 `maxAns`

## C++ 代码讲解

```cpp
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int sum = 0, maxAns = nums[0];
        for (int i = 0; i < nums.size(); ++i) {
            sum = max(sum + nums[i], nums[i]);
            maxAns = max(maxAns, sum);
        }
        return maxAns;
    }
};
```

关键点：

- `sum`：以当前下标 `i` 结尾的最大连续子数组和
- `maxAns`：目前见过的最大连续子数组和
- `sum + nums[i]`：把当前数接到前面的最佳连续子数组后面
- `nums[i]`：前面的和拖后腿时，从当前数重新开始
- `maxAns = max(maxAns, sum)`：答案可能出现在任意结尾位置，所以每一步都更新全局最优

循环不变量 / 状态含义：

- 遍历完 `i` 后，`sum` 是所有“以 `i` 结尾”的连续子数组中的最大和
- 遍历完 `i` 后，`maxAns` 是 `nums[0...i]` 中所有连续子数组的最大和
- 因为每一步只依赖上一步的 `sum`，所以可以把 DP 数组压缩成一个变量

边界处理：

- `maxAns` 必须初始化为 `nums[0]`，不能初始化为 `0`，否则全负数数组会错误返回 `0`
- `sum` 初始化为 `0` 配合从 `i = 0` 开始遍历是可行的，因为第一步会变成 `max(nums[0], nums[0])`
- 子数组至少包含一个元素，所以不能把空数组和当作候选答案
- `nums.length >= 1`，因此访问 `nums[0]` 是安全的

## 复杂度分析

- 时间复杂度：O(n)，数组只遍历一次
- 空间复杂度：O(1)，只维护 `sum` 和 `maxAns`

## 面试讲法

可以这样讲：

这题要求连续子数组，所以我考虑“以当前位置结尾”的最优解。遍历到 `nums[i]` 时，要么把它接在前一个位置结尾的最大子数组后面，要么前面的和已经是负贡献，就从当前元素重新开始。因此状态转移是 `sum = max(sum + nums[i], nums[i])`。因为最终答案不一定以最后一个元素结尾，所以我再用 `maxAns` 维护遍历过程中出现过的最大 `sum`。

可能追问：

- 为什么不是普通贪心：它可以看成 Kadane 算法，本质上也是压缩后的线性 DP
- 为什么负数时要重新开始：如果前缀和为负，它只会让后面的连续子数组变小
- 全负数怎么处理：`maxAns` 初始化为 `nums[0]`，所以会返回最大的那个负数
- 如果要返回区间怎么办：额外记录当前起点、最佳起点和最佳终点
- 分治怎么做：维护区间总和、最大前缀和、最大后缀和、最大子数组和，合并左右区间

## 模板沉淀

- 是否沉淀模板：是
- 对应模板：[以当前位置结尾的线性 DP](../templates/cpp/dynamic-programming.md)
- 可复用点：当题目要求“连续子数组/子串”的最值时，优先考虑以当前位置结尾的状态定义

## 易错点

- `maxAns` 不能初始化为 `0`，否则 `[-2, -1]` 会错
- `sum` 的含义不是“当前遍历过的总和”，而是“以当前位置结尾的最大子数组和”
- 更新顺序不能反：先更新 `sum`，再用新的 `sum` 更新 `maxAns`
- 子数组必须连续，不能跳过中间元素
- 子数组至少包含一个元素，不能用空子数组的和 `0` 参与答案

## 复盘卡片

- 一句话记忆：最大子数组和 = 每个位置决定“接上前面”还是“从自己重开”
- 下次先想：定义 `sum` 为“以当前位置结尾”的最大和，而不是全局和
- 如果写错，优先检查：`maxAns` 是否初始化为 `nums[0]`、是否先更新 `sum`、全负数是否正确
