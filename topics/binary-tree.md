# 二叉树

## 核心识别

什么时候想到二叉树套路：

- 输入是 `TreeNode* root`
- 需要访问每个节点，或根据左右子树关系做判断
- 题目要求前序、中序、后序、层序遍历
- 需要处理深度、高度、路径、祖先、子树等结构关系

## 常用套路

- BFS 层序遍历：用队列，按层处理时先固定当前层 `size`
- DFS 递归遍历：根据题意选择前序、中序或后序
- 自底向上：高度、平衡性、直径等题常用后序
- 路径搜索：递归维护当前路径或累计值

## 易错点

- 空树要先处理
- 按层输出时，必须区分当前层和下一层
- 访问 `node->left`、`node->right` 前，确认 `node` 非空
- 递归题要明确当前函数返回什么，遍历题要明确访问顺序

## 面试表达

二叉树题先判断是遍历问题、结构判断问题，还是路径问题。层序遍历优先用 BFS 队列；递归处理左右子树时，先说清楚当前节点要做什么、左右子树返回什么。
